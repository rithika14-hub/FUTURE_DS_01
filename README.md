# Business Sales Analytics Dashboard

## Project Overview
This project focuses on analyzing Superstore sales data to understand business performance, customer trends, sales patterns, and profitability using Python and Microsoft Excel.

## Objectives
- Analyze sales and profit performance
- Identify top-performing categories and regions
- Create interactive dashboards
- Generate business insights

## Tools & Technologies
- Python
- Google Colab
- Microsoft Excel
- Pivot Tables & Charts

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Dashboard Creation
4. Insight Generation

## Dashboard Features
- Sales Analysis
- Profit Analysis
- Category-wise Performance
- Region-wise Insights
- Interactive Slicers

## Key Insights
- High-performing regions contributed major sales
- Technology category showed strong growth
- Some products had low profitability despite high sales
- Seasonal trends impacted business performance
